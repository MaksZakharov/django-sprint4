# Blogicum — Blogging Platform

Веб-приложение для публикации постов с системой пользователей, комментариев и управления контентом.

## Описание проекта

Проект реализует блог-платформу с базовой системой управления контентом и пользователями.

Позволяет:
- создавать и публиковать посты
- управлять публикациями и комментариями
- работать с пользовательскими профилями
- разграничивать доступ к действиям

## Что реализовал

- Разработал веб-приложение на Django
- Реализовал модели данных:
  - пользователи и профили
  - посты, категории и местоположения
  - комментарии
- Реализовал бизнес-логику:
  - CRUD для постов и комментариев
  - отложенные публикации
  - отображение количества комментариев
- Настроил систему аутентификации (django.contrib.auth)
- Реализовал разграничение прав доступа (только автор может редактировать контент)
- Настроил пагинацию
- Реализовал CBV для страниц и логики приложения
- Настроил кастомные страницы ошибок (403, 404, 500)
- Настроил отправку писем (file-based backend)
- Написал автотесты (pytest)

## Основные возможности

- Регистрация и аутентификация пользователей
- Профиль пользователя и управление данными
- Создание, редактирование и удаление постов
- Работа с комментариями
- Категории и группировка контента
- Пагинация
- Отложенные публикации

## Технологический стек

Backend: Python 3.10+, Django  
База данных: SQLite (dev)  
Шаблоны: HTML, CSS  
Тестирование: pytest  

## Установка и запуск

```bash
git clone https://github.com/<your-username>/django-sprint4.git
cd django-sprint4
```

### Виртуальное окружение

```bash
python -m venv venv
source venv/bin/activate  # Linux / macOS
venv\Scripts\activate     # Windows
```

### Установка зависимостей

```bash
pip install -r requirements.txt
```

### Миграции и запуск

```bash
python blogicum/manage.py migrate
python blogicum/manage.py runserver
```

### (Опционально) загрузка данных

```bash
python blogicum/manage.py loaddata db.json
```

## Тестирование

```bash
pytest
```
